# もみかる採用LPプロジェクト：osteo.html を作成してください

## プロジェクト概要

リラクゼーションサロン「もみかる」の業務委託採用LPを作成します。今回はそのうち **接骨院・柔整・鍼灸師向け** の `osteo.html` を実装してください。

全体は7属性別LP＋ハブページの構成で、osteo.html は基準となる初号です。今後 single-mom / relax / esthe / housewife / side-job / owner の6本を同テンプレートで横展開するため、再利用しやすい構造で作成してください。

ターゲットの心理状態：
- 業務委託応募者は「会社の物語」ではなく「自分にとって損か得か」で判断する
- 国家資格者（柔整・鍼灸）は「修行という名の安売り」に強い不満を抱えている
- 0.3秒で「これは私のためのページだ」と認識させる必要がある
- スマホ流入80%超を前提

## アニメーション仕様

- 単一HTMLファイル。CSS・JSも全て同ファイル内（`<style>` `<script>` で内包）
- 外部依存は Google Fonts のみ
- `localStorage` `sessionStorage` は使用禁止
- Tailwind等のCSSフレームワーク不使用（CSS変数で統一）
- ストック画像・写真は使用禁止（権利問題）。すべて自作SVGで構成
- jQuery 不使用、素のJSで実装
- IntersectionObserver は必ず unobserve して再発火を防ぐ
- ヘッダーは scroll で `.scrolled` クラス付与（border-bottom が出る）
