1
こんな課題はありませんか？
✗ 新人研修はエーススタッフが担当するため、その間の施術売上がまるごと消える
✗ 研修中の売上ダウンがわかっているから、「短時間しか働けない人」を採用するとコストの無駄に感じてしまう
✗ エーススタッフの研修コストが重くのしかかり、週2〜3日勤務の短時間スタッフは採用に踏み切れない
→ これらすべて、Eラーニングシステムで解決できます

導入で変わる「3つのこと」
01教育コストの極小化
  • 40時間の対面研修を動画自習に置き換え、研修時間を半分以下に短縮
  • エーススタッフの研修時間を大幅削減。減った時間をそのまま施術売上に変えられる
  • 新人1人採用するごとに最大16万円のコスト削減効果（後述）
02採用戦略の劇的変化
  • 週2〜3日の短時間スタッフも利益の出る戦力として採用可能に
  • 「スマホで学べる最新研修」が求人での強力な差別化ポイントに
  • 採用母集団が広がり、慢性的な人手不足から脱却できる
03品質の均一化と向上
  • 座学・技術の流れを動画で予習・復習・テスト。格段に理解度が高まった状態で対面研修をスタート
  • 対面研修では手技のポイントや修正・検定だけに集中できる
  • デビュー後も動画を辞書として活用し、自己流化・品質劣化を防ぐ

導入後の研修フロー（3フェーズ）

テスト合格が実技参加の絶対条件です（例外なし）。もみほぐし → フット → ヘッド・ハンドの順に進みます。

PHASE 1　もみほぐし基本研修
①
アカウント作成
採用決定後、オーナー自身で管理画面からアカウントを発行。当日中にEラーニングを開始させる。
受講準備完了
②
Eラーニング受講
【リラクゼーション基礎知識】〜【もみほぐし全体の流れ】を動画で自習。自宅・移動中にスマホから受講可能。
言葉と流れを理解
③
テスト合格（必須）
【リラクゼーション基本手技】【もみほぐし全体手順】の両テストに合格。
⚠ 合格前に実技研修は開始しない。例外なし。
④
実技研修（短縮版）
流れ・用語は習得済みのため、インストラクターは手技の修正・ポイント・検定だけに集中できる。
従来 40時間 → 約20〜27時間 ▲最大20h削減
⑤
デビュー（もみほぐし）
実技検定合格でもみほぐしのデビューが可能。この時点から施術売上に貢献できる。
店舗デビュー
PHASE 2　フット オプション研修
⑥
Eラーニング ＋ テスト合格
フットの動画を視聴し手順テストに合格。もみほぐしと同じルールを適用（テスト合格が実技の絶対条件）。
実技参加権利発生
⑦
実技研修 → 合格
フット手技の修正・検定を対面で実施。
従来 10時間 → 約5〜7時間 ▲最大5h削減
PHASE 3　ヘッド・ハンド オプション研修
⑧
実技研修（動画補助）
研修時間が2〜3時間と短いためテストは実施しない。動画を補助的に活用しながら対面で実施。
ヘッド・ハンド習得
⑨
フルメニューデビュー
全メニュー対応が可能なスタッフとして稼働開始。デビュー後も動画を辞書として活用し、品質を維持。
フルデビュー完了
