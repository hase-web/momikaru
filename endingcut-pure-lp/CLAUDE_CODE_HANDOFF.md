# Claude Code 引き継ぎプロンプト集

## 1. 初回プロンプト（プロジェクト開始時に1回だけ実行）

ターミナルでLPプロジェクトのディレクトリに移動し、`claude`を起動した後、以下をそのまま貼り付けてください。

```
このプロジェクトはエンディングカットピュア合同会社のFC募集LPです。
ドラミカンパニー（代表：長谷川幸弘）が支援しています。

【ユーザールール】
- 「や」と「＃」の文字は使わない
- 同調ではなく、専門家・プロフェッショナルとして指摘・提案する
- 間違っていると思うことは理由とともに指摘
- 正しい点はさらに良いアイデアを見つけるためディスカッションを深める

【プロジェクト概要】
- 目的：説明会申込CVRの最大化
- ターゲット：独立志望者（経営者層・美容師・主婦・シニア）
- KPI最重要：説明会申込（フォーム送信）

【現在の状態】
index.htmlにLP本体があります。以下の構造で実装済み：

1. ヒーロー（映画的オープニング演出付き、約2.8秒で要素が時間差で現れる）
2. ご遺族の手紙（感情訴求）
3. 事業定義（エンディングカット技師®という新職業）
4. お悩みリスト（共感）
5. 中間CTA1
6. 市場性（年間死亡者数の推移チャート付き）
7. 6つの強み
8. 他社比較表
9. 収益シミュレーション（数字訴求）
10. 中間CTA2
11. オーナーの声（3名）
12. サポート体制（6項目）
13. Career Path（3段階＋トグル切替）
14. 開業ステップ
15. FAQ
16. 最終CTA（説明会限定特典3つ＋フォーム）
17. フッター＋フローティングCTA

【デザイン方針】
- 高級喪服ブランドのようなエディトリアル品位
- 配色：墨#0c0d10 × アイボリー#f4efe6 × くすみゴールド#b89968
- フォント：Shippori Mincho（見出し）、Cormorant Garamond（数字）
- セクション番号は漢数字でなくi. ii. iii.のラテン式

【Career Pathの3段階】
- step.i  1日養成講座（22,000円）
- step.ii 資格技師養成講座（55万円／110万円・トグル切替）
- step.iii FC加盟（A：県単位 660万/440万、B：市町村単位 220万）
※ロイヤリティ：総売上5%（利益の約5〜6%相当）
※段階間の進級割引はなし

【未確定・要確認事項】
- 加盟金の数字：シミュレーションでは220万、モデルケースでは300万と記載済（要統一）
- オーナーの声3名の数字は架空。実データへの差替が必要
- 利益率90%の根拠資料整備が必要（景表法対応）
- 写真素材は現行サイトのURLを参照中（差替推奨）

【次にやりたいこと】

まずindex.htmlを読み込んで、構造を把握してください。

## 2. 修正・追加依頼の定型プロンプト

### 軽微な修正
```
index.htmlの○○セクションで、△△を××に変更してください。
```

### 新セクション追加
```
index.htmlの○○セクションの後に、新しく□□セクションを追加してください。
- 目的：xxx
- 含める情報：xxx
- 既存のデザイン方針（明朝×ラテン×ゴールド）を踏襲
```

### 数字・コピーの差替
```
以下の確定情報で、index.html内の該当箇所を全て更新してください。
- 加盟金：◯◯万円
- 単価：◯◯円
- 利益率根拠：◯◯
（※確定値のみ伝えれば、Claudeが該当箇所を全部探して整合性をとってくれる）
```

### デザイン微調整
```
ヒーロー部分のオープニング演出を、もう少し□□にしてください。
- 全体の所要時間：現状2.8秒→○秒
- 要素の出現順：現状○○→希望は△△

## 4. CLAUDE.md（プロジェクトルートに配置するルールファイル）

ターミナルで以下を実行してCLAUDE.mdを作成してください。

```bash
cat > CLAUDE.md << 'EOF'
# CLAUDE.md - Endingcut Pure LP Project

## ユーザールール（必ず守ること）
- 「や」「＃」の文字は使わない
- 同調しない。専門家として間違いは指摘し、理由とともに代替案を提案する
- 正しい点はそこからさらに磨くディスカッションを行う

## プロジェクト概要
- クライアント：エンディングカットピュア合同会社（ドラミカンパニー支援）
- 目的：説明会申込CVRの最大化
- ターゲット：独立志望者・経営者・美容師経験者

## デザイン方針
- 配色：墨#0c0d10 / アイボリー#f4efe6 / くすみゴールド#b89968
- フォント：Shippori Mincho（見出し）、Cormorant Garamond（数字）
- 番号表記：i. ii. iii.（ラテン式）
- トーン：高級喪服ブランドのエディトリアル品位

## 主要セクション順序
Hero → Letter → Definition → Pain → MidCTA1 → Market → Strengths
→ Compare → Numbers → MidCTA2 → Voices → Support
→ CareerPath → Steps → FAQ → FinalCTA → Footer

## 重要な数字
- 単価：33,000円〜
- 利益率：最大90%（要根拠整備）
- 1日講座：22,000円
- 技師講座：550,000円（資格保有）／1,100,000円（資格なし）
- FC加盟金：A型660万/440万、B型220万
- ロイヤリティ：総売上5%

## 編集時の注意
- ヒーローはreveal classを付けない（独自アニメで動くため）
- Career Pathのトグルはstep.iiの料金・説明文を切り替える
- セクション番号（data-num）は01〜12で連続している
EOF
```

## 5. Git管理（推奨）

```bash
git init
echo "node_modules/
.DS_Store
*.log
" > .gitignore

git add .
git commit -m "initial: ECP LP base version with hero animation and career path toggle"
```

これで節目ごとに`git commit`を実行すれば、いつでも前のバージョンに戻れます。
