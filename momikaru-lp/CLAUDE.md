# CLAUDE.md — もみかるFC募集LPプロジェクト

## プロジェクトの目的

リラクゼーションサロン「もみかる」のフランチャイズ加盟検討者に向けたランディングページ（LP）を制作する。本LPの唯一のコンバージョンゴールは **資料請求フォームの送信獲得** である。

- 運営会社: 株式会社ドラミカンパニー
- 代表者: 長谷川幸弘
- ブランド: もみかる（2013年1号店開業、創業13年）
- 店舗数: 全国136店舗
- ターゲット: 30〜50代の事業家・独立志望サラリーマン
- 訴求軸: 信頼感・堅実・収益性

## 必須ルール

1. **DESIGN.md を必ず読むこと**
   - いかなる実装・編集タスクを開始する前にも、必ず `DESIGN.md` を最初に読み込み、記載されたビジュアル指針・数値基準に従うこと。
   - DESIGN.md に記載された数値（カラーコード、フォントサイズ、余白、ボタン高さ等）は変更せず、そのまま使用すること。

2. **数値遵守**
   - 本資料に記載された店舗数・創業年・年数・ターゲット属性などの数値・固有名詞を、推測で改変しないこと。
   - 新規に数値を追加する場合は必ずユーザーに確認を取ること。
   - 「多数の」「豊富な」など曖昧な表現の代わりに、確定した数値（例: 「全国136店舗」「創業13年」「事業継続率84%以上」「ロイヤリティ0円」）を用いること。

3. **自己検証**
   - ファイル作成・編集後、コントラスト比4.5:1以上、CTAボタン高さ56px以上、レスポンシブ動作などを自ら確認すること。
   - リンク切れ・閉じタグ漏れ・alt属性の有無を確認すること。
   - フォーム送信フローを目視確認するまで「完了」と報告しないこと。

4. **WCAG AA 準拠**
   - テキストと背景のコントラスト比は最低 4.5:1 を満たすこと。
   - タップ可能領域は最小 44×44px、主要CTAは最小高さ 56px を確保すること。

5. **禁止事項**
   - 絵文字の使用（ユーザーが明示的に指示した場合を除く）
   - DESIGN.md 不在での独自デザイン判断
   - 誇大広告表現・収益保証表現
   - 競合他社の固有名詞の無断使用

## 技術スタック

- HTML5（セマンティックマークアップ）
- Tailwind CSS CDN版（`<script src="https://cdn.tailwindcss.com"></script>`）
- Google Fonts: Noto Sans JP
- プレーンJavaScript（必要最小限、フレームワーク不使用）
- ビルドツール不要（CDN読み込みのみで完結）
