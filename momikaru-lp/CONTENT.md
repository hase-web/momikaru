## 5. 核となる4つの自動化（LPの論理構造）

本LPの骨格となる論理。Phase 4（解決策提示）の中核。

### 5.1 自動化①：人件費の自動化（完全歩合制）

| 項目 | 内容 |
|------|------|
| 仕組み | 固定人件費を変動費化、スタッフとは雇用契約ではなくパートナーシップ（業務委託等） |
| 効果 | 社会保険料負担ゼロ、退職リスクゼロ、煩雑な労務管理ゼロ |
| 業界課題 | 従来のリラクゼーションFC（従業員雇用型）は、店長人件費で利益が圧迫される |
| 結果指標 | 事業継続率 84%以上 |

### 5.2 自動化②：店舗運営の自動化（独自システムAndy）

**機能一覧**:
- 予約管理自動化（24時間WEB予約・POS連携）
- 会計・決済の一元管理（現金・カード・QR対応）
- シフト管理ワンクリック確定
- 複雑な歩合報酬計算の自動化（全銀データ対応）
- ダイナミックプライシング（土日祝自動価格UP）
- 自動追客・ステップメール（リピーター自動育成）

**業界課題**: 他FCはシステムが古く、管理業務は人力依存。
**結果**: 店長不要、オーナーはスマホで複数店舗管理可能。

### 5.3 自動化③：人材教育の自動化（Eラーニング）

| 項目 | 内容 |
|------|------|
| 仕組み | スマホ・タブレット対応クラウド型 |
| 内容 | 新人研修・接客マナー・施術技術・店舗運営ノウハウを動画網羅 |
| 管理 | 進捗管理機能でスタッフ学習状況をリアルタイム把握 |
| 業界課題 | 他FCは本部や直営店での集合研修が必要、時間・コスト大 |
| 結果 | 未経験者がゼロから自走してプロへ成長 |

### 5.4 自動化④：利益の自動化（スケールメリット）

**全プランロイヤリティ0円**を大前提とした上で、コアプランのシステム利用料逓減構造:

| 店舗数 | 月額システム利用料 | 減額率 |
|--------|------------------|-------|
| 1店舗目 | 15万円 | 基準 |
| 2店舗目 | 13.5万円 | −10% |
| 3店舗目以降 | 12万円 | −20% |

**業界課題**: 他FCは店舗数増加でロイヤリティ総額も増加、利益率が下がる。
**結果**: **店舗が増えるほど利益率が向上する、業界唯一の構造。**

## 10. 心理ストーリー設計（感情の動きと論理構成）

### Phase 1：認知（3秒で離脱させない）
「雇用リスクゼロで多店舗展開」という、既存常識を覆すメッセージで注目獲得。ロイヤリティ0円の事実を即座に提示する。

### Phase 2：共感（痛みの言語化）
以下の痛みに名前を付けて、読者に「わかってくれている」と感じさせる。

- 多店舗展開したいが、店長候補が採用できない
- 人件費で利益が消える
- 本業のリソースを割けない
- 2店舗目から採算が合わない
- 既存FCは結局儲からない

### Phase 3：業界の課題提示
- 従業員雇用型FCの限界
- ロイヤリティ負担の重さ
- 店長人件費による利益圧迫
- 属人性によるサービス品質のばらつき

→ 「業界全体の構造的問題」として提示。

### Phase 4：解決策の提示（4つの自動化）
「これは仕組みがない問題」と定義。4つの自動化（人件費・運営・教育・利益）で、全ての痛みが解消されることをロジカルに説明。

### Phase 5：証拠（数値と仕組み）
- 136店舗の全国展開
- 事業継続率 84%
- 独自システムAndyのスクリーンショット・仕組み図
- 医師監修のドクターズ・メソッド
- 商圏分析データシステム

→ 「仕組みがある」ことを具体的に証明。

### Phase 6：具体化（あなたに合うプランの提示）
3つのステップアッププランを並列提示。ベッドパートナー（0円スタート）で参入障壁を下げ、コアプラン（多店舗展開）で成長イメージを見せる。

### Phase 7：安心（リスクヘッジ）
協同組合・共済制度・商圏分析・本部サポートで、失敗リスクを多角的に低減。

### Phase 8：行動（説明会予約）
ハードルを下げて「まずは説明会で」と誘導。「実際の店舗のリアルな生データは説明会で全て公開」の訴求で、希少性・参加価値を演出。
